# チェックポイント機能

Gemini CLI には、AI ツールがファイルを変更する前にプロジェクトの状態を自動保存する「チェックポイント」機能があります。これにより、ツールを試した後でも元の状態にすぐ戻すことができます。

## 仕組み

ファイルシステムを変更するツールを実行すると、以下の情報が保存されます。

1. **Git スナップショット**: ホームディレクトリに作成される専用の Git リポジトリに、プロジェクトの現在の状態をコミットします。
2. **会話履歴**: その時点までの会話内容も保存されます。

チェックポイントはあなたの Git リポジトリには影響せず、いつでも状態を復元できます。
